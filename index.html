Pet Projects & Open-Source Contributions

I'm passionate about creating innovative solutions through personal projects. Here are some highlights:

  • BrowseRating↗

    A NextJS website that compares browser performance using the Speedometer 3.1 benchmark on identical hardware. Currently has over 160 stars on GitHub.

  • SafeTarot↗

    An intelligent fortune-telling and guidance application that provides personalized advice across various life topics. Users can interact with different AI-powered fortune teller personas, upload tarot card images for readings, and receive contextual guidance based on their emotional state and chosen topics.

  • Blackout↗

    A silent sleep aid for couples: a web app that helps a user drift off using ultra-dim, slow-changing text prompts (“micro-dream cues”) and minimal calming visuals—with no audio, no signup, and no scrolling. The core experience is a session player that fades away as the user falls asleep, so their partner remains undisturbed.

  • Binas↗

    A Node.js-powered real estate market analyzer for Tbilisi. It scrapes and categorizes data from Georgia's largest property listing website, providing valuable insights into average flat prices.

  • Celsiheit↗

    An interactive React application that visually demonstrates the relationship between Celsius and Fahrenheit temperatures. Perfect for educational purposes or quick conversions.

  • 5mdt Noise Generator↗

    A pure HTML and CSS solution for generating visual noise. I optimized the design for both mobile and desktop viewing, showcasing the power of CSS in creating dynamic visual effects.

  • Awesome Kagi CSS↗

    A curated collection of custom CSS themes for the Kagi search engine, featuring screenshots and implementation guides. The repository has earned over 45 stars on GitHub.
